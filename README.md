# Risk Of Tactics
Adds components, completed items and radiant items from TFT direct to your Risk of Rain 2 runs!  
![picture of items](https://i.imgur.com/nDgkbc8l.png)

## Mod Compatibility
Generally compatible with most mods. Supports integration with [LookingGlass](https://thunderstore.io/package/DropPod/LookingGlass/).

## DLC Support
Survivors of the Void DLC and Seekers of the Storm DLC are supported on all mod versions.

## Issue Reports
Please report any issues you find (bug, typo, mistranslation, incompatibility with another mod, etc.) on the [github issues tracker](https://github.com/shirograhm/RiskOfTacticsRoR2/issues). If possible, provide the output log as it often makes debugging much easier.

## Content
#### **Make sure to enable the "Use Custom Values" setting at the top of the config file, otherwise the mod will not respect your changes.**
The following values are configurable. To do so, run the game once with the mod installed, then navigate to the RiskOfTactics.cfg file, edit the values and relaunch the game.
If configuration issues persist, don't hesitate to delete the config file and let the mod regenerate it for you.
